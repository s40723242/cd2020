week13:

IPv4 遠端控制設定

操作步驟:

  1. 在cmd命令窗輸入ipconfig查IPV4 IP
  2. 將虛擬主機的主機IP改為個人IPV4 IP
  3. 啟動Ubuntu開啟exe
  4. 開啟putty，並設定IPV4 IP
  5. 設定SSH下的X11，將Enable X11 fowarding打勾並在下面對話窗中輸入localhost:0.0
  6. 存儲後再啟動即可連線，輸入kmol2020
  7. 輸入xterm&，cd cd2020pj1>git pull>cd docs/cmsimde>vi wsgi.py>更改host改為0.2.4，儲存
  8. 輸入python3 wsgi.py，網址為該主機IP加上17443(https://主機IP:17443

IPv6 遠端控制設定

更改內容:

network:
    ethernets:
        enp0s3:
           dhcp4: true
           dhcp6: true
           nameservers:
               addresses:
                  - 2001:b000:168::11  
  version: 2

操作步驟:

1.點選IPv6內容，IPv6位址輸入2001:288:6004:17:2020:2::7242(2為乙班、7227為乙班42號)
2.子網路首碼輸入64、預設匝道輸入2001:288:6006:17::254
3.開啟虛擬主機，設定系統埠號，將主機IP改為個人IP
4.啟動Ubuntu輸入密碼kmol2020
5.開啟LXTermial，並進入cd2020pj1
6.git pull檢查版本更新，cd docs/cmsimde
7.輸入vi wsgi.py，將host改為個人主機IP

8.在另一台主機開啟putty，輸入預連線主機IP
9.設定SSH下的X11，設定完成後存儲，在打開即可連線該主機
10.輸入帳密kmol2020，打開XLaunch.exe並輸入xterm&啟動
11.進入tmp/cd2020pj1/docs/cmsimde，輸入python3 wsgi.py開啟遠端
12.網址為[該主機IP]:17443
13.完成遠端控制

FileZilla 虛擬主機文件共享設定

操作步驟:

1.下載FileZilla

2.確認虛擬主機的IP後啟動

3.開啟putty，設定個人主機IP

4.設定SSH中的X11，打勾並輸入localhost:0.0存儲後開啟

5.輸入kmol2020

6.開啟FileZilla，點選檔案->站台管理員->新增站台

7.更改為 SFTP-SSH，輸入個人主機IP及22

8.更改為詢問密碼、命名為kmol2020，密碼輸入kmol2020

9.完成文件共享設定
