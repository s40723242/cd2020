以ssh對Github連線

影片:

步驟:

1.將E:\201906_fall\data\portablegit\bin中的sh_for_solvespace_compile複製並更名為sh.exe

2.進入黑窗輸入ssh-keygen -t rsa -b 4096 -C "使用者學號"

3.下載putty並透過putty.chm新增兩枚鑰使

4.修改start.bat:

set GIT_HOME=%Disk%:\portablegit\bin\
set GIT_SSH=%Disk%:\putty\plink.exe

5.修改自身倉儲 .git 中的 config 檔案中的 url 設定

url = git@github.com:mdecourse/cad2019.git

6.設定putty

7.在Y:\home_mdecourse\.ssh中新增config並加入指令

ProxyCommand y:/putty/plink.exe github.com %h %p
 
Host github.com
    User git
    Port 22
    Hostname github.com
 
    TCPKeepAlive yes
    IdentitiesOnly yes

8.在githun ssh設定中新增鑰使

9.完成

week12:

第四次開會紀錄:

IPv4 Ubuntu對外連線設定操作:

操作步驟:

  1. 喜好設定->網路->新增一個新的NAT網路
  2. 編輯NAT網路，新增4個連接埠轉送埠號
  3. 設定主機IP為0.0.1，主機連接埠號為 22、19999、17443、18443，客體IP為10.0.2.4，客體連接埠號為22、19999、17443、18443
  4. 點選設定>網路>設定為NAT Network
  5. 點選啟動，接著輸入密碼kmol2020
  6. 開啟LXTermial，輸入ifconfig> sudo apt install net-tools>密碼kmol2020
  7. cd tmp>ls>ping 127.0.0.1測試>cd cd2020pj1>git pull
  8. sudo vi wsgi.py編輯內容，將Host改成0.2.4
  9. 開啟leo，開啟cd cd2020pj1點NAV尋找allowExt
  10. 點選fileuploadfrom，在第10行新增一個ttt檔，存儲
  11. cd..，在tmp下新增oauth_scrum.txt並輸入sudo vi oauth_scrum.txt，儲存
  12. cd cd2020pj1>python3 wsgi.py
  13. 網址為https:/127.0.0.1:18443/alogin
  14. 帳號及密碼都為admin，進入後點選fileuploadform，點選要上傳的ttt
  15. 15.選download list檢查是否上傳完成
